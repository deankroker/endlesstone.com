Endless Tone

The Endless Tone Project

Endless Tone is focused on music-related product aggregation.

Aggregation

We use MongoDB with a distributed, RESTful search and analytics engine for product aggregation and querying. We allow for robust filtering using TypeScript: 

function ($stateProvider, $urlRouterProvider) {
  $urlRouterProvider.when('/products/','/products');
  $stateProvider
    .state('main.products', {
      url: '/products' +
      '?hand' +
      '?price' +
      '?make' +
      '?query' +
      '?sortBy' +
      '?sortType' +
      '?similar'+
      '?subCategory',
      templateUrl: 'app/modules/products/list/product.list.html',
      controller: 'ProductsController as productsCtrl',
    })
    .state('main.details', {
      url: '/details/:guitarId',
      templateUrl: 'app/modules/products/details/product.details.html',
      controller: 'ProductDetailsController as productDetailsCtrl'
    });
}

Ultimately, we were able to provide users with:

  • Robust matching capabilities amongst nine (9) different instrument categories.
  • Pricing for four (Amazon, Guitar Center, Sam Ash, Reverb) major new and used retailers.
  • Product description, specificatoin, and studio-quality images for fifteen (15) unique manufacturers.
  • The guaranteed best price on the web for each product listing.

Matching

Ensuring a system to maintain our product catalog was essential. It began with a series of functions, including:

  • Product scraping or API retrieval for each retailer
  • Word analysis for common brand and model retrieval
  • Match ruling based on brand-specific analysis
  • Product cleansing via Human Intelligence Tasks
  • One unified product listing for each brand and manufacturer
